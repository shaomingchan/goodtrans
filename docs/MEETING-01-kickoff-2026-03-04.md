## 二、项目定位

### 核心使命
**中文：** 愿世上没有文字的障碍  
**英文：** No language should be a barrier

### 产品愿景
打造世界最强 AI 反思翻译 SaaS，质量优于人工翻译。

### 目标用户
- 中小出海企业技术负责人
- 独立开发者
- 跨境电商卖家
- 内容创作者

### 核心差异化
1. **5轮反思翻译** - 不是机翻，是精翻
2. **支持100+语言** - Claude Opus 全语言支持
3. **中文场景优化** - 深度理解中文语境
4. **开发者友好** - API-first 设计
5. **透明定价** - 按字数计费，用多少付多少

---

## 三、定价策略

### 最终定价（统一标准）

**字数包定价：**
- **1万字包**：$250
- **5万字包**：$1,200（单价 $240/万字，5% 折扣）
- **10万字包**：$2,250（单价 $225/万字，10% 折扣）

**免费额度：**
- 普通注册：500 字（体验）
- 推荐注册：2000 字（体现专业度）

**推荐奖励：**
- 推荐用户充值，双方各得 10% 额度

### 定价逻辑
- 对标人工翻译主流价格的 75%
- 人工翻译：$25,000-42,000/百万字
- GoodTrans：$25,000/百万字（通用文档标准）
- 统一定价，不区分语言对和文档类型

## 四、技术方案

### 技术栈（基于 ShipAny 原模板）

**前端：**
- Next.js 14 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion（动画）

**后端：**
- Next.js API Routes
- 异步任务队列

**数据库：**
- Neon PostgreSQL
- Drizzle ORM

**认证：**
- Better Auth

**存储：**
- Cloudflare R2（译文存储）

**支付：**
- Creem（国内：支付宝/微信）
- Stripe（国际：信用卡/PayPal）

**部署：**
- Vercel

---

### 核心翻译流程（5轮）

**第 1 轮：PDF 提取**
- 工具：LlamaParse API
- 成本：$0.003/页
- 输入：PDF/Word/图片
- 输出：Markdown 格式文本

**第 2 轮：格式优化**
- 模型：Claude Sonnet 4.6
- 任务：清理冗余、标准化 Markdown
- Prompt：参考 Make 工作流

**第 3 轮：翻译初稿**
- 模型：Claude Sonnet 4.6
- 任务：专业翻译（信达雅标准）
- 支持：术语表自定义

**第 4 轮：翻译反思**
- 模型：Claude Opus 4.6
- 任务：评估质量，找出问题
- 输出：改进建议清单

**第 5 轮：翻译终稿**
- 模型：Claude Opus 4.6
- 任务：应用改进，输出最终译文
- 输出：高质量译文（Markdown）

---

### 支持的输入格式

1. **PDF** - LlamaParse 提取
2. **Word（.docx）** - 转 PDF 后处理
3. **Markdown（.md）** - 直接处理
4. **纯文本（.txt）** - 直接处理
5. **图片（.jpg/.png）** - OCR 提取（未来）

---

### 多语言支持

**IP 自动检测：**
- 使用 Vercel Geo API
- 日本 IP → 日文版
- 中国 IP → 中文版
- 欧美 IP → 英文版
- 其他国家 → 对应语言

**技术实现：**
- next-intl 国际化框架
- middleware 自动重定向

---

### 成本核算

**10万字翻译成本：**
- LlamaParse（100页）：$3
- Sonnet（2轮）：$7.5
- Opus（2轮）：$24
- **总成本：$34.5**

**毛利率：**
- 售价：$250/万字
- 成本：$34.5/10万字
- **毛利率：86.2%**

## 六、用户体验设计

### 核心流程（3步完成）

1. **上传文档** - 拖拽上传，即时报价
2. **配置翻译** - 选择语言，开始翻译
3. **下载结果** - 对比查看，下载译文
