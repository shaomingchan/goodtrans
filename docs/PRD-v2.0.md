## 一、产品概述

### 1.1 产品定位

GoodTrans 是一款基于 AI 反思翻译技术的 SaaS 产品，致力于提供接近人工翻译质量的专业翻译服务。

**核心使命**：
- 中文：**专业翻译，机器速度，人工质量**
- 英文：**Professional Translation. Machine Speed. Human Quality.**

**产品愿景**：
打造最优质的自动化翻译工具，让翻译不再是障碍。

### 1.2 核心价值主张

**相比传统机器翻译（DeepL/Google Translate）**：
- ✅ 5 轮反思翻译，质量更高
- ✅ 三层术语库系统，术语高度一致
- ✅ 支持 100+ 语言
- ✅ 按需付费，无需订阅

**相比人工翻译**：
- ✅ 速度快 100 倍（10 万字 1 小时 vs 1-2 周）
- ✅ 成本低 3-8 倍（$0.025/字 vs $0.1-0.3/字）
- ✅ 术语高度一致（人工翻译约 70%）
- ✅ 质量稳定（AI 不会疲劳）
- ✅ 无限规模化
- ✅ **全自动化**（免去分段翻译的痛苦）

**核心差异化**：
- **自动化**：上传文档 → 留下邮箱 → 1小时后收到译文
- **反思翻译**：5轮迭代优化，不是简单的一次性翻译
- **术语库**：三层术语系统，确保专业术语一致性

## 三、核心功能

### 3.1 功能架构

```
┌─────────────────────────────────────────┐
│           用户界面层                     │
│  - 文档上传                              │
│  - 邮箱输入                              │
│  - 支付确认                              │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│           业务逻辑层                     │
│  - 文档解析（MinerU）                    │
│  - 术语提取（自动）                      │
│  - 翻译引擎（5轮反思）                   │
│  - 邮件通知（自动发送）                  │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│           数据存储层                     │
│  - 用户数据（PostgreSQL）                │
│  - 文件存储（Cloudflare R2）             │
│  - 术语库（PostgreSQL）                  │
│  - 任务队列（Inngest）                   │
└─────────────────────────────────────────┘
```

### 3.2 核心功能详解

#### 功能 1：5 轮反思翻译引擎

**第 1 轮：PDF 提取**
- 工具：MinerU（开源 PDF 解析）
- 输入：PDF/Word/图片
- 输出：Markdown 格式文本
- 时间：5-30 秒

**第 2 轮：格式优化**
- 模型：Claude Sonnet 4.6
- 任务：清理冗余、标准化 Markdown
- 时间：5-10 秒

**第 3 轮：翻译初稿**
- 模型：Claude Sonnet 4.6
- 任务：专业翻译（信达雅标准）
- 支持：强制使用术语表
- 时间：10-20 秒/段

**第 4 轮：翻译反思**
- 模型：Claude Opus 4.6
- 任务：评估质量，找出问题
- 输出：改进建议清单
- 时间：10-20 秒/段

**第 5 轮：翻译终稿**
- 模型：Claude Opus 4.6
- 任务：应用改进，输出最终译文
- 输出：高质量译文（Markdown）
- 时间：10-20 秒/段

**总时间**：
- **10 万字约 1 小时**（CEO 决策：用户可以接受）
- 技术方案：3并发 + Inngest 任务队列

#### 功能 2：三层术语库系统

**第一层：用户私有术语库（最高优先级）**
- 用户自定义术语
- 强制使用
- 支持导入/导出

**第二层：官方术语库（中等优先级）**
- 按行业分类（技术/医学/法律/金融/工程）
- 预置 10 万条术语
- 持续更新

**第三层：众包术语库（最低优先级）**
- 用户贡献的术语
- 社区投票验证
- 未来功能

**术语提取流程**：
1. 反向匹配：在文档中搜索术语库中的术语（1 秒）
2. AI 补充：Opus 识别新术语（30 秒）
3. **自动应用**：不需要用户确认（CEO 决策）
4. 翻译集成：每段翻译都使用术语表

**术语一致性承诺**：
- **MVP 阶段**：实测 100 个文档后再定
- **不承诺 100%**（CEO 决策：不夸大宣传）
- 目标：>95%

#### 功能 3：全自动化流程（CEO 核心要求）

**用户体验**：
1. 上传文档
2. 输入邮箱
3. 支付确认
4. **关闭页面**（不用等待）
5. 1小时后收到邮件（附带下载链接）

**邮件模板**：
```
主题：您的翻译已完成 - GoodTrans

您好，

您的文档翻译已完成！

- 原文：[文件名]
- 字数：10,000 字
- 翻译时间：45 分钟
- 质量评分：92/100

点击下载：[下载链接]（有效期 7 天）

如有问题，请回复此邮件。

GoodTrans 团队
```

**技术实现**：
- Inngest 任务队列（异步处理）
- Resend 邮件服务
- R2 临时下载链接（7天有效期）

#### 功能 4：质量评估

**自动检查项**：
1. 术语一致性（实测后确定目标）
2. 格式完整性（段落、标题、列表）
3. 数字准确性（日期、金额、百分比）
4. 专有名词保留（人名、地名、公司名）

**质量报告**：
- 术语使用统计
- 质量评分（0-100）
- 潜在问题标记

#### 功能 5：多语言支持

**支持语言**：100+ 语言（Claude Opus 全语言支持）

**重点语言**：
- 中文 ↔ 英文
- 中文 ↔ 日文
- 中文 ↔ 韩文
- 英文 ↔ 西班牙文
- 英文 ↔ 法文

### 3.3 支持的文件格式

| 格式 | 支持状态 | 处理方式 |
|------|---------|---------|
| PDF | ✅ MVP | MinerU 提取 |
| Word (.docx) | ✅ MVP | 转 PDF 后处理 |
| Markdown (.md) | ✅ MVP | 直接处理 |

| 图片 (.jpg/.png) | 🔄 Phase 2 | OCR 提取 |
| Excel (.xlsx) | 🔄 Phase 3 | 表格翻译 |

## 四、用户体验设计

### 4.1 核心流程（3 步完成）

**步骤 1：上传文档**
- 拖拽上传或点击选择
- 支持 PDF/Word/Markdown/TXT
- 即时显示字数和报价
- 选择目标语言

**步骤 2：输入邮箱并支付**
- 输入邮箱地址
- 选择字数包（1万/5万/10万字）
- 支付确认（Creem/Stripe）

**步骤 3：等待邮件通知**
- 显示预计完成时间（如：1小时后）
- 用户可以关闭页面
- 翻译完成后自动发送邮件
- 邮件包含下载链接

### 4.2 砍掉的功能（CEO 决策）

**不做流式预览**：
- 理由：技术实现太复杂，会导致系统臃肿
- 替代方案：邮件通知 + 下载链接

**不做实时进度展示**：
- 理由：用户不会盯着屏幕等1小时
- 替代方案：显示预计完成时间

**不做术语表确认**：
- 理由：增加用户负担
- 替代方案：自动提取并应用

### 4.3 设计风格

**参考**：Squarespace 简约风格
- 大量留白
- 简洁排版
- 专业配色

**技术栈**：
- shadcn/ui 组件库
- Tailwind CSS 4

## 七、竞争分析

### 7.1 核心护城河（CEO 强调）

**最大护城河：自动化**
- 免去客户分段翻译的痛苦
- 上传 → 邮箱 → 1小时后收到译文
- 不是每个人都能实现自动化的反思翻译

**第二护城河：术语库**
- 三层术语系统
- 官方术语库 + 用户自定义
- 确保专业术语一致性

**第三护城河：5轮反思翻译**
- 不是简单的一次性翻译
- 反思 → 改进 → 终稿
- 质量接近人工翻译

### 7.2 竞争优势

| 维度 | DeepL | Google | 人工翻译 | GoodTrans |
|------|-------|--------|---------|-----------|
| 翻译质量 | 7/10 | 6/10 | 8/10 | 接近 8/10 ✅ |
| 术语一致性 | 70% | 60% | 70% | >95% ✅ |
| 处理速度 | 快 | 快 | 慢 | 1小时 ✅ |
| 成本 | 订阅制 | 免费 | 高 | 按需付费 ✅ |
| **自动化** | ❌ | ❌ | ❌ | ✅ |
| 反思翻译 | ❌ | ❌ | ❌ | ✅ |
